## MotoGP Results Manager
A MotoGP stats-driven data manager, managed by CSV dataset files from Kaggle.
Made by Carlos Carvajal (@cacara82)

## Usage
1.- You will need poetry to run this project: poetry install
2.- Update the imports: poetry update
3.- You can now run "app/main.py" to start

## Important
Do NOT rename the dataset files, as the project uses their default names to load
them into memory. If you do, when you fetch new data you will have file duplicates with
different names.
